# 🚴 Bike Buyers Data Analysis (Excel)

## 📌 Project Overview

This project focuses on cleaning, analyzing, and visualizing a Bike Buyers dataset using Microsoft Excel. The objective was to transform raw data into meaningful insights through data cleaning, pivot tables, charts, and an interactive dashboard.

---

## 🎯 Objectives

* Clean and prepare raw data for analysis.
* Identify trends and patterns in customer purchasing behavior.
* Build pivot tables to summarize key metrics.
* Create charts and a dashboard for easy data visualization.

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboard
* Excel Formulas

---

## 🧹 Data Cleaning

The following cleaning steps were performed:

* Removed duplicate records
* Trimmed unnecessary spaces (`TRIM`)
* Standardized text formatting (`PROPER`)
* Corrected inconsistent values
* Checked for missing or invalid data
* Organized the dataset for analysis

---

## 📊 Analysis Performed

The cleaned dataset was analyzed to understand:

* Bike purchase trends
* Customer demographics
* Income distribution
* Age group comparisons
* Marital status analysis
* Regional comparisons

The analysis was summarized using Pivot Tables and visualized through Pivot Charts.

---

## 📈 Dashboard

An interactive Excel dashboard was created to present key insights in a clear and user-friendly format.

Dashboard features include:

* Pivot Charts
* Slicers for filtering data
* Key performance summaries
* Easy-to-understand visualizations
